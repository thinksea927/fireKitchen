# Fire Kitchen 
> 在忙碌的上班日子中，你是否已經忙到無法再開伙？  
知道要吃的健康，但卻只能匆匆的挑選便利商店的減醣便當。

我們的主廚 Antony，又稱「Fire Antony」。  
精準的掌握火侯且熟知每樣食材最美味的秘密，  
每樣餐點都經過設計，給你低碳健康又吃不膩。
